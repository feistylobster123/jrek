# JREK - Western States 100

A QWOP-style ragdoll physics running game featuring Scott Jurek running the Western States 100 mile endurance race.

## Play

Open `index.html` in any modern browser. No build tools or server needed.

## Controls

- **J** - Left thigh (hip joint)
- **R** - Right thigh (hip joint)
- **E** - Left calf (knee joint)
- **K** - Right calf (knee joint)
- **SPACE** - Start / Restart

## About

Scott Jurek won Western States 7 consecutive times (1999-2005). This game lets you experience the difficulty of coordinating legs to run 100 miles from Squaw Valley to Auburn, California.

Getting even 10 feet is an achievement. Getting a mile would be legendary.

## Tech

- Matter.js for physics
- HTML5 Canvas for rendering
- No frameworks, no build tools, no npm. Just static files.
